HomeAssistant, ESPHome und eine Matrix Uhr

Mar 9, 2023

Vor knapp einem Jahr habe ich die abgebildete „Uhr“ nachgebaut. Die Idee habe ich von Qromes marquee-scroller. Von Ihm stammt auch das gedruckte Gehäuse, ich habe lediglich komplett eigenen Code geschrieben.

Meine erste Version zeigt die Uhrzeit an und verfügte über eine sehr einfache HTTP API. Über diese API besteht die Möglichkeit, einen Text über das Display scrollen zu lassen oder Animation ablaufen zu lassen. Das ganze lässt sich über einen einfachen iOS Shortcut bedienen.

Das ganze habe ich für meine Kids gebaut - bei jedem steht nun diese Uhr im Zimmer.

Version 2

Seit kurzem habe ich zu Hause HomeAssistant laufen, mit ESPHome wollte ich immer schon mal spielen und somit habe ich die Uhr nun mit ESPHome geflasht. Mit einer einfachen Konfiguration (esphome-matrix-clock.yaml) lässt sich das gute Stück nun in HomeAssistant einbinden und zeigt nun die Uhrzeit an.

Mit HomeAssistant ist nun alles mögliche machbar.
